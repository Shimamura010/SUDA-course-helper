# SUDA-course-helper
苏大双创教育平台刷课插件——自动播放 · 跳过题目 · 连播功能（职业生涯规划课程可用）

# 苏大双创教育平台刷课插件

> ⚠️ **免责声明**: 本项目仅供学习交流使用，请勿用于商业用途或违反学校规定的行为。使用者需自行承担相关责任。

## 功能特性

- 🎯 自动视频播放与进度管理
- ⏩ 智能跳过题目检测
- 🔄 自动连播下一章节
- 🔊 永久静音模式
- ⚡ 可调节播放速率
- 🕒 时长伪造与进度模拟
- 🖥️ 实时状态监控面板

## 安装使用

### 前置要求
- 浏览器安装 Tampermonkey 或 Violentmonkey 脚本管理器

### 安装步骤
1. 点击本项目的 `suda-course-helper.user.js` 文件
2. 点击 "Raw" 获取原始文件
3. Tampermonkey 会自动检测并提示安装
4. 点击安装即可

### 使用方法
1. 登录苏大双创教育平台
2. 进入课程视频页面
3. 插件会自动运行，右上角会出现控制面板
4. 根据需要调整设置

## 配置选项

在脚本开头可以修改以下配置：
```javascript
const config = {
    playbackRate: 1.0,        // 播放速率
    autoSkip: true,           // 自动跳过题目
    autoContinue: true,       // 自动连播
    permanentMute: true,      // 永久静音
    // ... 更多配置
};


> ⚠️ **重要提示**: 本脚本必须在浏览器扩展的**开发者模式**下使用，否则可能无法正常运行。

## 🛠️ 安装前必读

### 开发者模式要求
由于这是一个用户脚本，**必须**在浏览器的开发者模式下安装和使用：

#### Chrome/Edge 浏览器设置：
1. 打开 `chrome://extensions/` 或 `edge://extensions/`
2. **开启右上角的"开发者模式"**
3. 然后安装 Tampermonkey
4. 在 Tampermonkey 中安装本脚本

#### Firefox 浏览器设置：
1. 打开 `about:addons`
2. 进入扩展管理
3. **启用开发者模式**
4. 安装 Tampermonkey 和本脚本

### 为什么需要开发者模式？
- 用户脚本需要扩展管理器的高级权限
- 确保脚本能够正确拦截和修改网页请求
- 避免浏览器安全策略阻止脚本运行
